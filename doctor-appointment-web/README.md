# Doctor Appointment Web App

## Setup
1. Create the folder structure as above.
2. Add placeholder images to `images/` (e.g., download free stock photos for doctors/hero).
3. Open `index.html` in a browser.
4. For production: Host on GitHub Pages or Vercel.

## Customization
- Update `doctors` array in `script.js` with Figma data.
- Export colors/fonts from Figma to match exactly.
- Add more pages (e.g., `about.html`) if the design has them.

## Features
- Responsive design (mobile/desktop).
- Search & filter doctors.
- Booking modal with form validation.
- Smooth animations/transitions.
